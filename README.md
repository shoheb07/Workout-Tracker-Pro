# Workout-Tracker-Pro
Workout Tracker Pro is a Python-based desktop application that helps users track workouts, monitor body weight, calculate BMI, estimate calories burned, and visualize fitness progress through interactive charts. Built with Tkinter, SQLite, and Matplotlib, it provides an easy-to-use interface for managing personal fitness data.

Workout Tracker Pro

Workout Tracker Pro is a desktop fitness application built with Python, Tkinter, SQLite, and Matplotlib. It enables users to log workouts, track body weight, calculate BMI, estimate calories burned, and visualize progress through charts.

Features

- Add and manage workout records
- Track body weight over time
- Calculate Body Mass Index (BMI)
- Estimate calories burned for different activities
- View progress charts and statistics
- Search and delete workout entries
- Export workout data to CSV
- Personal records tracker
- Dark mode support
- Backup and restore database

Tech Stack

- Python
- Tkinter
- SQLite
- Matplotlib
